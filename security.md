<summary>🔧 ❌ 🛠️ 常见问题与解决方案</summary>

### ❌ 虚拟机无法启动，提示「VT-x 禁用」

**原因：** BIOS 中未开启虚拟化技术。

**解决方案：**
1. 重启电脑，进入 BIOS/UEFI（通常按 F2、Del 或 F12）
2. 找到「Intel VT-x」或「AMD SVM」
3. 设置为「Enabled」
4. 保存并重启

### ❌ 拖拽文件无效

**原因：** VMware Tools 未安装或未更新。

**解决方案：**
1. 启动虚拟机
2. 菜单「虚拟机」→「安装 VMware Tools」
3. 在客户机中运行安装程序
4. 重启虚拟机

### ❌ 虚拟机卡顿

**原因：** 分配的 CPU/内存不足。

**解决方案：**
1. 关闭虚拟机
2. 右键虚拟机 →「设置」
3. 增加 CPU 核心数和内存大小
4. 建议：Windows 10/11 分配 4GB 以上，Linux 分配 2GB 以上

</details>

## ❓ 常见问题

**❓ 这是真的破解版吗？**  
是的。完全免费，VMware Workstation Pro 全部功能解锁。

**❓ 和免费版 VMware Player 有什么区别？**  
Player 只能运行现有虚拟机，不能创建、不能快照、不能克隆、不支持多个快照。Pro 版全部功能都有。

**❓ 支持 macOS 虚拟机吗？**  
技术上可以，但需要额外解锁补丁（unlocker）。默认 VMware 不支持 macOS。

**❓ 玩游戏卡吗？**  
3D 加速支持 DirectX 11，玩老游戏没问题。新 3A 大作建议双系统或实体机。

**❓ 安全吗？**  
是的。VMware 是全球顶级软件，数千万用户使用。激活密钥来自合法渠道。

**❓ 支持哪些客户机系统？**  
Windows、Linux、BSD、Solaris、DOS、macOS（需解锁）。几乎支持所有 x86 系统。
